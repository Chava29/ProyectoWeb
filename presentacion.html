Liga de Fútbol

Este sistema web permite administrar una pequeña liga de fútbol usando páginas JSP, conexión JDBC y una base de datos en MySQL. Desde aquí podrás registrar equipos, guardar los resultados de los partidos y generar una tabla de posiciones automática.

¿Qué puede hacer el sistema?

  • Registrar nuevos equipos con su nombre, ciudad y estadio.
  • Guardar los partidos jugados entre dos equipos y sus goles.
  • Mostrar la lista de equipos y de partidos registrados.
  • Calcular la tabla de posiciones con partidos jugados, goles y puntos.

Tecnologías utilizadas

  • HTML / CSS para la interfaz.
  • JSP para la lógica del servidor.
  • JDBC para conectar con MySQL.
  • MySQL como gestor de base de datos.
